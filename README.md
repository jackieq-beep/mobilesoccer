# Mobile Soccer Media Network (GitHub Pages)

## Upload
1) Create a GitHub repo (or reuse your existing one)
2) Upload ALL files in this folder (keep the folder structure)
3) Settings → Pages → Deploy from branch → (main) / (root)

## What was fixed
- Desktop: shows the normal nav links, hamburger hidden.
- Mobile (<=900px): shows hamburger, desktop links hidden.
- Mobile menu only appears when hamburger is tapped (no more duplicated links).
